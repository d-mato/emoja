# Emoja

## Usage

```ruby
Emoja.translate("🍎食べたい")
# => "赤リンゴ食べたい"

Emoja.search("猫")
# => ["😺", "😸", "😹", "😻", "😼", "😽", "🙀", "😿", "😾", "🐱", "🐈", "🐾"] 
```

## Installation

```ruby
gem 'emoja'
```
